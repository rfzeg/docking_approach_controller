## Description

- Minimal docking approach controller node (Python)


## Instructions

Build with:  

`colcon build`   

After compiling, source the workspace otherwise you will get the "Package 'docking_approach_controller' not found" error message:  

`source install/setup.bash`  



## Dependencies
- ROS2 Galactic
